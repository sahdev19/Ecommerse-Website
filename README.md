# E-Commers-by-sahdev
Sahdev-Ecommerse Website - A fully responsive online shopping platform featuring 8 products from popular brands. Includes a functional shopping cart system with add/remove items, real-time price calculations, user authentication with sign-in page, mobile-friendly navigation, and smooth scrolling. Built with HTML, CSS, and JavaScript using sessionStorage for cart persistence and login state management. Key Features: Product catalog, shopping cart, user login, responsive design, smooth animations, and persistent data storage. Tech Stack: HTML5, CSS3, JavaScript, Font Awesome Icons.
